# Fraud Detection System

A **machine learning project** to detect fraudulent credit card transactions in real-time using **LightGBM** and a **Streamlit web app**.

## Features

- Detects fraudulent transactions in real-time.  
- Handles categorical and numerical features.  
- Uses transaction distance as a key feature.  
- Balances dataset with SMOTE.  
- User-friendly Streamlit interface.

---

## Workflow

1. **Data Preprocessing:** Handle datetime, encode categorical features, create new features.  
2. **Data Analysis & Balancing:** EDA and SMOTE for class imbalance.  
3. **Model Training:** LightGBM classifier.  
4. **Evaluation:** Classification report, ROC-AUC, feature importance.  
5. **Deployment:** Streamlit app for real-time predictions.
